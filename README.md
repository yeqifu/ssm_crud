# ssm_crud
基于spring+springmvc+mybatis的增删改查(来自：尚硅谷雷丰阳)
### 一、技术选型：
#### 1.基础框架：spring+springMVC+mybatis
#### 2.前端框架：BootStrap3
#### 3.数据库：mysql5.0
#### 4.项目管理工具：maven3.5.2
#### 5.分页插件：PageHelper
### 二、功能介绍：
#### 可以实现员工的增删改查与分页。
